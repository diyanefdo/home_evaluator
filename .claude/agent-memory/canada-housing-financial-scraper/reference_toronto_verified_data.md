Verified data points (retrieved 2026-06-25) and the sources that reliably carry them. Use for Toronto/GTA scenarios; re-verify currency before reuse.

- Toronto 2026 residential property tax total rate: 0.7673% (municipal + education + city-building). Source: toronto.ca property-tax-rates page; corroborated catax.tools, bremo.io.
- Best 5yr fixed (Jun 2026): ~3.94% (best), insured ~4.04%, Big-6 discounted avg ~4.93%, best 5yr variable ~3.35%. Source: ratehub.ca, wowa.ca, nesto.ca.
- Toronto long-term home appreciation: ~7% CAGR since 1978; 1996-2021 = 7.08% CAGR. But 2026 is a correction year (detached avg ~$1.36M, down ~4.7% YoY). Sources: rates.ca, remax.ca, precondo.ca, TRREB.
- S&P 500 30yr nominal CAGR with dividends (to May 2026): 10.31%; real (inflation-adj) 7.57%. Source: slickcharts.com/sp500/returns, macrotrends.
- Reliable URLs: CMHC HMIP (www03.cmhc-schl.gc.ca/hmip-pimh) for rent/HPI tables; TRREB market-data/housing-market-charts; Statcan table 34-10-0145-01 for 5yr conventional mortgage history; zumper.com/rent-research for current neighbourhood rent.
- Data gap: postal-code-level (M2J/North York) price & rent rarely isolated; default to GTA/Toronto CMA or neighbourhood (Zumper) figures and note substitution.
